# 第一步

1. 安装 node.js

2. 安装 typescript

```JavaScript
npm init -y //初始化

npm  i typescript -g
tsc -v //查看版本

// 初始化
tsc --init

// nodejs 环境执行ts

npm i ts-node -g //
npm i @types/node -D // 安装nodejs 类型定义

// 可以采用 实时监听我们的 ts 文件，然后生成 js 文件
tsc -w


```
